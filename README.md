# 🧠 CogniCore // AI Web Hub

CogniCore is a high-efficiency academic workspace and neural chat engine built to optimize study workflows, log lecture metadata, and generate contextual study assets. Powered by advanced Large Language Models via Groq, the platform features a reactive dual-input terminal that accepts both standard text and real-time vocal transmissions.

---

## 🛠️ System Core Features

* **📝 Core Data Workspace:** An interactive data matrix entry window allowing users to compile and commit subject-specific lecture notes into an active runtime memory layer.
* **💬 Neural Chat Engine (Voice + Text):** A dual-input communication stream integrated with **Groq's Whisper-Large-v3** for real-time voice capture processing and **Llama-3.1-8b-instant** for instant contextual feedback.
* **⚡ AI Compilation Tools:** An automated academic synthesis matrix capable of generating:
    * Structural bulleted lecture summaries
    * 5-tier Q&A interactive flashcard pairs
    * 3-question multiple-choice evaluations with automated answer keys
    * Tactical day-by-day exam roadmaps based on customizable buffer thresholds.
* **🗂️ Notebook Persistence Matrix:** An active sidebar panel equipped with live global filtering query nodes to seamlessly jump between stored repositories.

---

## 🧬 Technology Stack Architecture

* **Runtime Environment:** Python 3.14+
* **Web Framework Node:** Streamlit
* **Inference Engine Network:** Groq Cloud API (Llama 3.1 & Whisper v3)
* **Environment Security:** Python-Dotenv Matrix
