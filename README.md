# 🧠 CogniCore // AI-Powered Study Desktop Engine

CogniCore is a modern, dark-themed Python desktop application engineered with `customtkinter` and integrated with the lightning-fast **Groq Llama-3.1 API**. It bridges the gap between raw student lecture notes and interactive machine-learning study tools.

## ⚡ Core Systems Architecture
* **Live AI Chat Stream:** Instantly query complex scientific topics using an advanced LLM terminal.
* **Algorithmic Summarizer:** Strips text down to high-efficiency bullet points instantly.
* **Flashcard & Quiz Compiler:** Dynamically tracks context from your notes to generate evaluation mock-tests.
* **Persistent Local Database:** Seamlessly serializes and parses internal states via JSON storage.

## 🛠️ Installation & Deployment
Ensure you have Python 3.10+ installed, then clone the repository and run:

pip install customtkinter groq
python app.py
